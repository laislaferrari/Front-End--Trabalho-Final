# SerraPet - Portal de adoção e caridade destinado à animais resgatados

O **SerraPet** é uma plataforma interativa desenvolvida como projeto final da disciplina de Front-End Essencial. Foi desenvolvida uma página para conectar animais que precisam de um lar a famílias responsáveis, além de um bazar beneficente e um portal de doações diversas.

## Tecnologias Utilizadas

O projeto foi construído utilizando práticas do mercado e tecnologias ensinadas durante as aulas realizadas durante o programa de residência do Serratec.

- **HTML5 Semântico:** Estruturação de conteúdo focada em SEO e acessibilidade.
- **CSS3 Personalizado:** Estilização utilizando variáveis CSS e metodologia **BEM** (*Block Element Modifier*).
- **Bootstrap 5:** Sistema de Grid (12 colunas) e componentes responsivos.
- **JavaScript (ES6+):** Manipulação dinâmica do DOM, uso de Template Literals e métodos de array.
- **Acessibilidade:** Validado com a ferramenta **WAVE** (Web Accessibility Evaluation Tool).

## Funcionalidades Principais

- **Catálogo Dinâmico:** Listagem de cães e gatos gerada via JavaScript.
- **Efeitos Visuais:** Ícones decorativos flutuantes com animações de `:hover` e `transition`.
- **Design Responsivo:** Adaptado para telemóveis, tablets e computadores.
- **Página "Quem Somos":** História e missão da equipa integradas via sistema de ficheiros.
- **Página de bazar beneficente, página de formulários de adoção e apadrinhamento e de doações diversas;**
